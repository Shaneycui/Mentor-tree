## 项目简介

MentorTree 是一个专注于医疗行业的职业发展平台，帮助医疗行业从业者探索职业路径、寻找导师指导、参与行业交流。

## 主要功能

### 1. 职业路径探索
- 医疗实体企业组织架构展示
- 医疗金融机构岗位体系展示
- 交互式职业发展路径图
- 岗位详细信息查看

### 2. 导师咨询
- 导师筛选和匹配
- 在线预约咨询
- 导师评价系统
- 专业领域分类

### 3. 问答社区
- 行业问题讨论
- 标签分类系统
- 搜索功能
- 问答互动

### 4. 个人中心
- 个人信息管理
- 学习记录追踪
- 咨询历史查看
- 问答记录管理

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Flow
- Supabase
